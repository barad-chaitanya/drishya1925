# 🌸 Drishya's Birthday Site

## How to add your photos

1. Rename your photos to `photo1.jpg`, `photo2.jpg` ... `photo12.jpg`
2. Drop them all into the `photos/` folder
3. Done — the gallery picks them up automatically

You can add more or fewer photos by editing the `PHOTOS` array in `app.js`.

---

## How to deploy on Render (free)

1. Push this folder to a GitHub repo (can be private)
2. Go to [render.com](https://render.com) → **New → Static Site**
3. Connect your GitHub repo
4. Set:
   - **Build command:** *(leave blank)*
   - **Publish directory:** `.` (just a dot)
5. Click **Deploy** — Render gives you a free `.onrender.com` URL

That's it. No server, no backend, completely free forever on the free tier.
